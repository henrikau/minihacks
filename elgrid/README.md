# electricity cost

Simple script for grabbing daily values from Noordpool and display a
summary of the days prices.

## Help
``` bash
cost.py -h
usage: cost.py [-h] [-f FILE] [-c CITY]

Track packages via Posten/Bring's API

optional arguments:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  Store .js-file to parse (for testing/dev)
  -c CITY, --city CITY  Only display city
```
